# Description

eMAG Promoted Remover automatically removes promoted and sponsored product listings from eMAG search results and category pages.

Supports emag.ro, emag.bg, and emag.hu.

Features:
- Instantly hides promoted product cards as pages load
- Continuously monitors for dynamically inserted promoted listings
- Tracks removal statistics — see how many ads are blocked per page and across your lifetime usage
- Clean, minimal footprint with only the storage permission required

No data is collected or sent anywhere. All statistics are stored locally in your browser.
